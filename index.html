    Pesquisa sobre formulários:

    1. Na tag <input> com atributo type="text", para que serve o atributo "placeholder"?
      • O atributo placeholder é utilizado para podermos inserir uma sugestão de valores no campo de inserção, podemos ter como exemplo uma sugestão de valor para o usuário antes dele inserir propriamente os valores. no caso do texto vai existir uma sugestão de texto que servirá de exemplo para o usuário.
    2. Na tag <input> com atributo type="radio" ou type="checkbox", para que serve o atributo "checked"?
      • Serve para definir uma caixa de seleção quando marcada dentre as outras opções. As caixas de seleção são usadas para que o usuário possa selecionar uma ou mais opções dentre as caixas que estão disponíveis para ele.
    3. Na tag <input> para que serve o atributo "value"?
      • A tag atributo valor serve para atribuir um determinado valor á um elemento, entretanto o atributo valor é usado de forma diferente para cada tipo de entrada, por isso não podemos usar por exemplo <input type=''file''>
    4. Que atributo deve ser utilizado para agrupar vários botões de rádio em um formulário de modo que apenas um deles fique marcado de cada vez?

      • O atributo rádio deve compartilhar o mesmo atributo “name” para, desta forma, ser tratado como um grupo. Os botões de opção são normalmente apresentados em grupos de rádio (uma coleção de botões de opção que descrevem um conjunto de opções relacionadas). Apenas um botão de opção em um grupo pode ser selecionado ao mesmo tempo.

    5. Para que serve a tag <label>? Para que serve o atributo "for" nessa tag?

      • A “label” (rótulo) serve para informar para que serve cada campo do formulário HTML, facilitando o entendimento do usuário. Se você utilizar a tag <label> juntamente com o parâmetro “for”, poderá fazer um vínculo entre um rótulo e um campo do formulário de forma que, ao clicar no rótulo, você terá o mesmo resultado como se clicasse no campo vinculado. Para que o parâmetro “for” da tag <label> funcione corretamente, devemos também definir o parâmetro “id” do controle relacionado.

    6. Na tag <option> para que serve o atributo "selected"?
      • A <option>tag define uma opção em uma lista de seleção
      • <option>elementos vão dentro de um elemento <select> , <optgroup> ou <datalist> .
      • O atributo “selected” é um atributo booleano. Se utilizado, especifica que uma opção deve ser pré-selecionada quando a página for carregada. A opção pré-selecionada será exibida primeiro na lista suspensa. Também pode ser definido após o carregamento da página, com um JavaScript.
      • Nota: A <option>tag pode ser usada sem nenhum atributo, mas normalmente você precisa do atributo value , que indica o que é enviado ao servidor no envio do formulário.
    7. Nas tags de formulário para que serve o atributo "required"
      • O atributo “required” especifica que um campo de entrada deve ser preenchido antes de enviar o formulário. Funciona com os seguintes tipos de entrada: text, search, url, tel, email, password, date pickers, number, checkbox, radio e file.
    8. Na tag <form> para que serve o atributo "action"?
      • O atributo "action" especifica para onde enviar os dados do formulário quando um formulário é enviado. Normalmente, os dados do formulário são enviados para um arquivo no servidor quando o usuário clica no botão enviar.
    9. Na tag <form> para que serve o atributo "method"?
      • O atributo "method" especifica o método HTTP a ser usado ao enviar os dados do formulário. Os dados do formulário podem ser enviados como variáveis ​​de URL (com method="get") ou como transação de postagem HTTP (com method="post").
      • O método HTTP padrão ao enviar dados de formulário é GET, o qual não deve ser usado para dados confidenciais, pois ficam visíveis na URL e é limitado a 2048 caracteres. Já no POST, se os dados do formulário contiverem informações confidenciais ou pessoais, estas não são mostrados na URL. O POST não tem limitações de tamanho e pode ser usado para enviar grandes quantidades de dados.
    10. Para que o valor digitado de uma senha não apareça na barra de endereços do navegador, qual deve ser o valor do atributo "method" na tag <form>?

    11. Qual valor deve ser informado no atributo "method" na tag <form> em um formulário que será usado para fazer upload de arquivos?
